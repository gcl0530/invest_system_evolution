# 投资助手小程序

## 项目概述
一款专为个人投资者设计的微信小程序，涵盖基金/股票行情关注、自定义策略提醒、金字塔加仓管理、投资体系沉淀四大核心功能。

## 页面结构（4页 + tabBar）

| 页面 | 路径 | 功能 |
|------|------|------|
| 基金 | pages/fund | 基金关注列表、净值展示、K线+技术指标 |
| 股票 | pages/stock | 股票关注列表、行情展示、K线+技术指标 |
| 策略 | pages/strategy | 自定义买卖策略、金字塔加仓计划、触发提醒 |
| 体系 | pages/philosophy | 道法术器势投资框架整理与沉淀 |

## 技术指标

### K 线图组件 (`components/kline-chart`)
- **均线**: MA5 / MA10 / MA20 / MA60
- **布林带**: BOLL(20, 2)
- **MACD**: DIF / DEA / 柱状图 (12, 26, 9)
- **RSI**: RSI(14)
- **KDJ**: K / D / J (9, 3, 3)
- **成交量**: VOL + 5日均量

### 指标计算引擎 (`utils/indicators.js`)
- SMA / EMA / BOLL / MACD / RSI / KDJ 全部本地计算
- Canvas 2d 渲染，支持触摸交互
- 可切换显示/隐藏各指标

## 策略引擎 (`utils/strategy.js`)

### 触发条件类型
1. 均线交叉 (price_cross_ma)
2. MACD金叉/死叉 (macd_cross)
3. 布林带突破 (boll_breakout)
4. RSI超买超卖 (rsi_over)
5. 放量信号 (volume_surge)
6. 目标价格 (price_target)

### 金字塔加仓模型
- 4层金字塔：试探仓(10%) → 基础仓(20%) → 加仓1(30%) → 加仓2(40%)
- 自动计算每层股数、资金、平均成本
- 止损/止盈价格自动计算
- 支持自定义每层比例

## 投资体系 - 道法术器势

| 层级 | 含义 | 说明 |
|------|------|------|
| 道 | 投资哲学 | 核心理念、价值观、能力圈 |
| 法 | 投资体系 | 策略框架、仓位管理、买卖纪律 |
| 术 | 技术分析 | 趋势判断、买卖信号、量价关系 |
| 器 | 工具平台 | 行情工具、数据分析、交易执行 |
| 势 | 市场环境 | 经济周期、货币政策、行业景气 |

## 数据层

当前使用 `mock/data.js` 模拟数据，实际部署需接入：
- 基金数据：天天基金/腾讯财经 API
- 股票数据：东方财富/新浪行情 API
- 策略触发：后端定时检查 + 微信订阅消息推送
